前言與警語

歡迎使用質借試算工具！本手冊將引導您如何有效地使用本網站提供的各項試算功能。

重要警語：此網站為作者做為試算練習使用，不保證獲利，不保證數字的正確性。使用者請自行斟酌使用。所有計算結果僅供參考，實際投資可能面臨市場風險、利率風險等多種不確定因素。使用者應對自己的投資決策負責。建議在做出任何投資決策前，諮詢專業財務顧問意見。

當您首次訪問入口頁面時，會彈出警語視窗，請仔細閱讀並點擊「我已了解並同意」後方可繼續使用。此同意狀態會記錄在您的瀏覽器中，除非清除瀏覽器資料，否則不會重複顯示。

入口頁面操作 (index.html)

這是網站的主頁，提供兩個主要試算工具的入口：

  • 疊券試算：點擊「進入試算」會導向 beautified-bond-calculator.html。
  • 疊基金試算：點擊「進入試算」會導向 beautified-fund-calculator.html。

頁面下方提供「使用說明」和「版本更新歷程」。

點擊任一試算工具前，若您尚未同意警語，系統會先請您同意。

疊券試算工具 (債券與基金質借收益率評估表)

此工具用於詳細評估多筆債券質借以及一筆房貸的綜合現金流與風險狀況。

參數設定 (匯率、借款成本)

在頁面頂部，您可以設定以下全域參數：

  • 各幣別兌台幣匯率 (USD/TWD, AUD/TWD, 等)：
    • USD/TWD：此為主要計算用匯率，您可以手動輸入。
    • 其他幣別 (AUD, CAD, CNY, EUR, GBP) 兌 TWD 的匯率欄位為唯讀，會透過API自動更新。
  • 債券借款成本 (%)：輸入您預估的債券質借年利率。預設為 2.95%。
  • 房貸借款成本 (%)：輸入您的房屋貸款年利率。預設為 2.60%。
  • 房屋貸款金額(台幣)：輸入您的房貸總額。預設為 30,000,000。

操作按鈕：

  • 強制更新匯率 (API)：點擊此按鈕會強制從外部 API 獲取最新的各幣別匯率，並更新對應欄位（包括 USD/TWD）。
  • 更新參數/匯率：
    • 此按鈕會將您在「債券借款成本」、「房貸借款成本」及「房屋貸款金額」輸入框中修改的值應用到計算中。
    • **匯率更新邏輯**：如果您手動修改了 USD/TWD 欄位，此按鈕將保留您的手動輸入；否則，它會嘗試從 API 更新所有匯率。
    • 點擊後，所有相關統計數據會重新計算。

提示：匯率欄位旁的 會顯示API匯率的更新狀態。

新增/編輯債券

  • 點擊 新增債券 按鈕彈出對話框。
  • 填寫債券資訊：商品代碼、名稱、幣別 (USD, AUD, CAD, CNY, EUR, GBP)、標普評等、追繳維持率、剩餘年期、參考買入價 (系統會自動+3作為實際買入價)、投資面額、質借成數、利率(%)、配息頻率、下次配息日。
  • 點擊「儲存」。債券會出現在下方表格中。
  • 表格中每行債券後方有 編輯 和 刪除 按鈕。
  • 可以拖曳表格中的 圖示來調整債券順序。

注意：「投資金額」是根據「投資面額」和「參考買入價(+3)」自動計算的市值。「質借金額」是根據「投資金額」和「質借成數」自動計算的。

從Excel導入資料 (更新現有債券)

  • 點擊 導入資料 按鈕，選擇一個 .xlsm 格式的 Excel 檔案。
  • 工具會嘗試讀取名為「熱門債券」的工作表 (若找不到則讀取第一個工作表)。
  • 它會根據「商品代碼」匹配您在表格中已有的債券，並用 Excel 中的數據更新這些債券的：名稱、幣別、標普評等、剩餘年期、參考買入價 (自動+3)、票息、配息頻率、下次配息日。
  • 此功能僅更新已存在於計算器表格中的債券，不會從 Excel 新增債券。

從篩選工具導入 (新增債券)

如果您從「債券篩選工具」匯出了選定的債券：

  • 當您打開此「疊券試算」頁面時，系統會檢測到來自篩選工具的數據，並顯示一個通知。
  • 點擊通知中的 導入 按鈕。
  • 系統會詢問您是否要清除現有債券列表。選擇「確定」以清除並導入，選擇「取消」則在現有列表後附加新導入的債券。
  • 導入的債券會使用計算器頁面上當前設定的「債券借款成本」。

您也可以點擊 前往篩選工具 按鈕直接跳轉到篩選工具頁面。

查看計算結果

頁面下方會顯示關鍵的計算結果：

  • 整戶維持率 (%)：(總投資金額市值 / 總質借金額) * 100%。
  • 整戶追繳維持率 (%)：通常為銀行要求的最低維持率，例如 113%。
  • 整戶追繳跌幅 (%)：您的資產可以承受多大的跌幅而不被追繳。
  • 每年配息(台幣)：所有債券年配息（換算台幣） + 所有基金年配息（台幣）。
  • 每年借款成本(台幣)：所有債券年借款成本（換算台幣） + 房貸年借款成本（台幣）。
  • 扣除利息每年現金流(台幣)：每年配息(台幣) - 每年借款成本(台幣)。

點擊 重新計算 按鈕可以隨時刷新這些統計數據。

其他操作 (清除、列印)

  • 清除數據：清除所有已新增的債券和基金資料。
  • 列印：調用瀏覽器的列印功能，列印當前頁面內容。

疊基金試算工具 (基金質借收益率評估表)

此工具專為評估多層基金/債券質借策略設計，特別關注第一層質借後的再投資情況。

參數設定

  • 匯率：影響整體現金流換算（如果您的基金投資或配息涉及外幣轉換到台幣，但此工具主要以台幣計）。預設為 32。
  • 基金/債券借款成本 (%)：您質借基金或債券時的年利率。預設為 2.95%。
  • 房貸借款成本 (%)：您的房屋貸款年利率。預設為 2.60%。
  • 房屋貸款金額(台幣)：您的房貸總額。預設為 30,000,000。

點擊 更新參數 按鈕，會將上述輸入框中的值應用到計算中，並重新計算所有統計數據。

新增/編輯基金/債券

  • 點擊 新增基金/債券 按鈕。
  • 在彈出對話框中：
    • 搜尋基金/債券：可輸入代碼或名稱進行模糊搜尋，系統會提供匹配選項。
    • 商品類型：選擇「基金」或「債券」。
    • 商品代碼/名稱：可從搜尋結果選擇，或手動輸入。
    • 追繳維持率 (%)：銀行要求的最低維持率。預設 140%。
    • 投資金額(台幣)：輸入該商品的投資總額（已換算為台幣）。對於第二層以後的投資，這通常是上一層質借出來的金額。
    • 質借成數 (%)：該商品可質借的百分比。預設 45%。
    • 利率 (%)：該商品的年化配息率或票面利率。
    • 配息頻率：選擇月配、季配、年配或不配息。
  • 點擊「儲存」。商品會出現在下方表格中。
  • 表格中每行商品後方有 編輯 和 刪除 按鈕。
  • 可以拖曳表格中的 圖示來調整順序（順序對此工具的特定計算邏輯很重要）。
  • 每行商品後方有 查看 按鈕，可連結至國泰世華銀行基金資訊頁面 (基於商品代碼)。

重要提示 (疊基金邏輯)：

  • 此工具的核心計算（如整戶維持率、追繳跌幅、基金借款成本）主要基於您新增的**第一筆**基金/債券的數據。
  • 後續新增的基金/債券主要用於累計「每年配息」，其質借金額不直接計入「整戶」的借款成本和維持率計算，而是假設其投資金額來源於上一層的質借。
  • 因此，請確保表格中的**第一筆商品**是您主要的質借標的。

查看計算結果

頁面下方會顯示關鍵的計算結果：

  • 整戶維持率 (%)：基於第一筆商品的 (投資金額 / 質借金額) * 100%。
  • 整戶追繳維持率 (%)：通常為銀行要求的最低維持率，例如 140%，或您為第一筆商品設定的值。
  • 整戶追繳跌幅 (%)：基於第一筆商品的數據計算，顯示資產可承受多大跌幅而不被追繳。
  • 每年配息(台幣)：所有新增基金/債券的年配息總和（台幣）。
  • 每年借款成本(台幣)：第一筆商品的年借款成本 + 房貸年借款成本（台幣）。
  • 扣除利息每年現金流(台幣)：每年配息(台幣) - 每年借款成本(台幣)。
  • 扣除利息每月現金流(台幣)：每年現金流 / 12。

點擊 重新計算 按鈕可以隨時刷新這些統計數據。

其他操作 (清除、列印)

  • 清除數據：清除所有已新增的基金/債券資料。
  • 列印：調用瀏覽器的列印功能。

債券篩選工具 (beautified-bond-tool.html)

此工具協助您從 Excel 檔案中篩選符合特定條件的債券。

上傳 Excel 檔案

  • 點擊「選擇檔案」按鈕或將 Excel 檔案 (.xlsm, .xlsx) 拖曳到指定區域。
  • 工具會自動嘗試讀取名為「熱門債券」的工作表。若無此工作表，則讀取檔案中的第一個工作表。
  • 工具會自動從指定欄位讀取數據：距到期年 (AA欄)、下次配息日 (AJ欄)、幣別 (M欄)。其他欄位 (如發行人、買入價等) 會嘗試從表頭關鍵字動態匹配。

設定篩選條件

檔案成功載入後，您可以設定以下篩選條件：

  • 發行人：輸入關鍵字篩選。
  • 幣別：輸入幣別代碼 (如 USD, AUD) 篩選。
  • 標普評級：選擇比較運算子 ( =, >=, <=, >, < ) 並輸入評級 (如 AAA, BBB-)。
  • 距到期年：可設定兩個條件，例如大於X年且小於Y年。
  • 參考買入價：選擇比較運算子並輸入價格。
  • 票息 (%)：選擇比較運算子並輸入票息率。
  • 下次配息日：選擇特定日期。

點擊 套用篩選 執行篩選，或點擊 重置 清除所有篩選條件。

查看與匯出結果

  • 符合條件的債券會顯示在下方的表格中。
  • 您可以點擊表頭進行排序。
  • 勾選您想要的債券。可以使用 全選 或 取消全選 按鈕。
  • 點擊 匯出選定債券。這會將選中的債券數據儲存到您瀏覽器的暫存空間中。
  • 然後，點擊 開啟計算器，這會打開「疊券試算工具」頁面。該頁面會自動檢測到您匯出的數據並提示導入。

通用提示

  • 所有試算結果僅供參考，不構成任何投資建議。
  • 請務必根據自身風險承受能力和財務狀況做出決策。
  • 百分比輸入：在輸入利率、成本率、質借成數等百分比時，請直接輸入數字部分，例如輸入 5.5 代表 5.5% (除非欄位旁已標示 %)。
  • 數據保存：
    • 「疊券試算」和「疊基金試算」中的數據（您新增的債券/基金）只會保存在當前瀏覽器會話中。關閉瀏覽器或分頁後數據可能會遺失。
    • 「債券篩選工具」匯出的數據是暫存在瀏覽器中，用於傳遞到「疊券試算工具」。
  • 遇到問題或顯示異常，嘗試刷新頁面或點擊各工具的「重新計算」按鈕。
